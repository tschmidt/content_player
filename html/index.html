jQuery Plugin: Content Player

Description

A light-weight plugin that allows you to create a slideshow for HTML content.

Requirements

This plugin requires jQuery 1.6+ and jQuery UI 1.8+

Options

Option Description Default
autoPlay [true, false] Specifies whether the "slideshow" should play automatically. false
autoPlayInterval Specifies the length of time the slideshow will wait before advancing to the next slid. (Only applicable if autoPlay is set to true) 7000ms
autoPlayStopWhenClicked Specifies whether the slideshow should stop if you click one of the navigational elements true
autoPlayRestartAfterInterval Specifies the time to wait before the slideshow resumes. (Only applicable if autoPlayStopWhenClicked is set to false) 15000ms
locationHashSupport Supports using hash in the URL to set the initial slide to show true
dynamicArrows Creates "next" and "previous" arrows for navigation if set to true true
dynamicArrowsPrevText Sets the text that will be displayed for the previous navigation link (Only applicable if dynamicArrows is set to true) "&#171; prev"
dynamicArrowsNextText Sets the text that will be displayed for the next navigation link (Only applicable if dynamicArrows is set to true) "next &#187;"
dynamicTabs Creates tabs for each "slide". The title for the tab is determined by the element with the class set by tabTitleSelector. true
dynamicTabsPosition ["top", "bottom"] Determines whether tabs appear on top of the slideshow or the bottom. top
tabTitleSelector The class that will be used to find the element who's text will be used for the tab title (see examples for more information). .title
firstSlideToLoad Specifies the initial slide that should be active. 1 (specifies the first slide in the set)
slideEaseDuration Specifies the time it will take to transition between slides 1000ms
slideEaseFunction Specifies the easing function that will used during the slide transition (see jQuery Easing Plugin for a list of options) jswing

HTML Structure

One of the things that I don't like about similar plugins is the strict HTML structure or extraneous markup that you need to use in order for the slideshow to work. In my approach to this problem I did my best to make sure the plugin creates any HTML element that it will require leaving you, the creator of your content, responsible for making the markup as semantic as possible.

With that said, there is one tiny bit of extra markup that you will need. Unfortunately, I have not found a way around this and as such you will simply need to wrap your "slides" in a block level container. 

<div id="slideshow1" class="slideshow-container">
  <!-- Your "slides" here -->
</div>

And that's it. The id and class you choose to give the container is completely up to you. One more thing to note is that there is no default CSS that comes with this plugin. You can use what I have done here for inspiration, but again, I chose to leave the CSS up to you.
